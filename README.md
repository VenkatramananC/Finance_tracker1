# Finance Tracker

A web-based finance tracking application built with Flask and SQLite.

## Features
- User authentication with role-based access (admin, analyst, viewer)
- Add and manage income/expense transactions
- View transaction history and summaries
- Secure password hashing

## Tech Stack
- Python, Flask
- SQLite
- HTML, CSS

## Setup
1. Install dependencies: `pip install -r requirements.txt`
2. Run: `python app.py`
3. Open: `http://localhost:5000`
